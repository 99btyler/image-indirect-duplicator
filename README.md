# Getting Started
Get this project set up locally
### Prerequisites
* Pillow (`pip3 install Pillow`)
### Setting up
* Clone this repository
* Using a text editor, open (repository)/src
### Running
* Using terminal, navigate to src and type `python3 main.py`
